# Evaluate a news article with Natural Language Processing (NLP): Udacity

## description

tool to run Natural Language Processing (NLP) on articles

## Installation

```bash
npm install
```

if You are using yarn

```bash
yarn install
```

## Usage

In the project directory, run:

use dev server

```bash
npm run build-dev
```

production version

```bash
npm run build-prod
```

then

```bash
npm start
```

testing

```bash
npm run test
```
